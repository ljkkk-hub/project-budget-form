## 项目立项预算表网页工具
- 基于用户提供的 Excel 模板（项目预算表格 副本6.xlsx），创建了单 HTML 文件的网页填写工具
- 功能：主管在线填写立项金额和 KPI 数据，一键导出与模板格式一致的 Excel 文件
- 技术栈：纯前端（HTML + Tailwind CSS + SheetJS），无需后端
- 包含两个核心区域：KPI 指标表（支持动态增删行）和成本预算表（6类成本×12月）
- 实时计算和校验（权重合计=100%、金额合计=合同收入）
- Excel 导出还原模板格式：合并单元格、边框、列宽、公式、黄色填充
